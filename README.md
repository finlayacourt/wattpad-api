# wattpad-api
Unofficial API for searching wattpad stories

## Installation
```shell
$ git clone https://github.com/finlayacourt/wattpad-api.git wattpad-api
$ cd wattpad-api
$ npm install
```
> Don't have `npm` installed? Get it from [here](https://www.npmjs.com/get-npm)
## Start
```shell
$ npm start
```
